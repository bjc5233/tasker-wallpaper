# tasker-wallpaper



## 说明
通过tasker下载\处理图片  
处理图片为了方便需要tasker插件AutoShare  
目前可透露情报：处理图片目前存在生成图片质量略低问题，主要是[tasker-common](https://github.com/bjc5233/tasker-common)中的[comm_base64_img]模块导致，持续改进
